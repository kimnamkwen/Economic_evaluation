# 보건의료 빅데이터 및 비교효과연구 경제성평가 센터

이 프로젝트는 센터 및 연구실 소개, 임상연구, 빅데이터 분석, 경제성 평가, 프로젝트, 강의 정보 제공을 위한 웹 포털 프론트엔드입니다.

## 주요 화면 기능

- 센터 및 연구진 소개
- 임상연구·빅데이터 분석·경제성평가 기본 이론 안내
- 진행중/완료 프로젝트 및 강의 일정 안내
- 논문, 보고서, 템플릿 등 다운로드 연동
